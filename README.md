Overview
This is a simple Restaurant Management System developed in Python. The project allows users to select food items from different categories such as Starters, Main Course, Desserts, and Drinks. It calculates the total bill based on the selected items and quantities.
Features
Menu-driven interface
Multiple food categories
Starters
Main Course
Desserts
Drinks
Quantity selection for each item
Automatic price calculation
Final bill generation

How It Works
User selects a starter item and quantity.
User selects a main course item and quantity.
User selects a dessert item and quantity.
User selects a drink and quantity.
The program calculates the total bill and displays the final amount.
